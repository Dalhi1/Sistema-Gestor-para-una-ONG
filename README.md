  # Como correr el codigo

    abrir la carpeta donde descagarste y descomprimiste el .rar en cmd ejemplo cd C:\Users\PC\Documents\pp

    escribir npm install

    ya que se instalen las dependencias escribir npm run dev

    va a aparecer un link de localhost, abrirlo en chrome o edge

    despues de la primera vez que instalaste las librerias para volver a abrirlo solo sera necesario abrir la ruta y ejecutar npm run dev
